Sobre mi

Las palabras que mejor me definirían son: Desarrollador frontend que sabe maquetar y diseñar. Mi nombre es Julio Martínez, llevo diseñando y maquetando paginas web desde 2007. A lo largo de estos años he podido desarrollar mi rol como diseñador / maquetador / programador front en varias empresas; unas veces en el departamento de Marketing, formando parte del equipo creativo y haciendo de nexo de unión entre Marketing e IT y otras en el departamento de IT como desarrollador de la parte más visual de numerosos sitios.


UI/UX Designer & Frontend Developer.

Mi especialidad es la maquetación responsive, diseños en html 5, CSS 3 y javascript que se adapten a todas las resoluciones y espacios.
El framework CSS y javascript que más uso es Bootstrap y el framework de desarrollo Angular. Me gusta añadir animaciones CSS muy sutiles a los diseños y uilizar imagenes svg escalables a todas las resoluciones. Cuido hasta el último detalle para que la maquetación se corresponda 100% con el diseño.
